# 프로젝트 번역 용어집

이 프로젝트(Learn Claude Code)의 핵심 용어 번역 기준. translator와 reviewer가 공통으로 참조한다.

## 핵심 용어

| 영어 | 한국어 | 중국어 | 일본어 | 비고 |
|------|--------|--------|--------|------|
| Harness | 하네스(Harness) | 框架(Harness) | ハーネス(Harness) | 원어 병기 필수 |
| Agent | 에이전트 | 智能体 | エージェント | |
| Agent Loop | 에이전트 루프 | 智能体循环 | エージェントループ | |
| Tool Use | 도구 사용 | 工具使用 | ツール使用 | |
| Subagent | 서브에이전트 | 子智能体 | サブエージェント | |
| Context Window | 컨텍스트 윈도우 | 上下文窗口 | コンテキストウィンドウ | |
| Context Compaction | 컨텍스트 압축 | 上下文压缩 | コンテキスト圧縮 | |
| Skill | 스킬 | 技能 | スキル | |
| Task System | 태스크 시스템 | 任务系统 | タスクシステム | |
| Background Task | 백그라운드 태스크 | 后台任务 | バックグラウンドタスク | |

| Autonomous Agent | 자율 에이전트 | 自主智能体 | 自律エージェント | |

| Prompt | 프롬프트 | 提示词 | プロンプト | |

## 번역하지 않는 용어

다음은 원어 그대로 사용한다:
- API, CLI, SDK, JSON, YAML, MCP
- Claude, Claude Code, GitHub, Git
- Python, TypeScript, JavaScript, React, Next.js
- pip, npm, pnpm
- README, CLAUDE.md

## 용어 추가 원칙

1. 기존 번역물(docs/zh/, docs/ja/)에서 이미 사용된 번역어를 최우선으로 채택
2. 해당 언어 개발자 커뮤니티에서 가장 널리 쓰이는 번역어 선택
3. 1, 2가 모호하면 원어 병기 형태 사용
